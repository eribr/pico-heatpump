API Configuration

This interface communicates with the Pico Heat Pump API through a secure PHP bridge.

  • All commands require authentication
  • Connection is encrypted via HTTPS/SSL
  • API bridge runs with elevated security

Temperature Range

Operating temperature: 10°C to 32°C

API Endpoints

Available endpoints:

  • GET /api/status - Get current status
  • PUT /api/power/{on|off} - Control power
  • PUT /api/mode/{auto|heat|cool|dry|fan} - Set operating mode
  • PUT /api/temp/{10-32} - Set temperature
  • PUT /api/fan/{auto|silent|2|3|4} - Set fan speed
  • PUT /api/swing/{auto|swing|up|mup|middle|mdown|down} - Set swing
